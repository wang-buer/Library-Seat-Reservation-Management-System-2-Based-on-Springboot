# 基于springboot的图书馆座位预约管理系统2



## 1、项目介绍

基于springboot和vue的图书馆座位预约管理系统2拥有三种角色

管理员：座位管理、学生管理、签到管理、离开信息管理、信用分管理等

学生：预约座位、暂离、离开、签到等

教师：新增座位、审核、查看各种信息等

具体功能请观看功能视频演示

## 2、项目技术

后端框架：springboot、Mybatis

前端技术：VUE、html

## 3、开发环境

- JAVA版本：JDK1.8（最佳）
- IDE类型：IDEA、Eclipse都可运行
- 数据库类型：MySql（5.7、8.x版本都可） 
- tomcat版本：无需
- maven版本：无限制
- 硬件环境：Windows
